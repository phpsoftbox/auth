# CLI

## Синхронизация ролей

```
php psb auth:role:sync
```

Команда синхронизирует роли/пермишены из definitions с БД.
Во время синхронизации удаляются роли и пермишены, которых нет в definitions.

### Dry-run

```
php psb auth:role:sync --dry-run
```

Показывает, какие роли и пермишены будут добавлены/удалены, без записи в БД.

## Просмотр прав роли

```
php psb auth:role:permissions admin
```

Показывает список разрешённых и запрещённых пермишенов для конкретной роли.

## Назначение роли пользователю

```
php psb auth:role:assign 42 manager
```

Команда назначает пользователю указанную роль.

## Снятие роли с пользователя

```
php psb auth:role:discharge 42 manager
```

Команда снимает указанную роль с пользователя.
